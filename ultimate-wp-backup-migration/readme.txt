== Installation ==

1. Upload the plugin files to `/wp-content/plugins/ultimate-wp-backup-migration/`
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Setup Wizard to configure your backup system
4. Start creating backups!

== Frequently Asked Questions ==

= Does this plugin have file size limits? =

No! Unlike other backup plugins, Ultimate WordPress Backup Migration has no artificial file size restrictions. It can handle sites of any size.

= Do I need to pay for additional features? =

No subscriptions or additional purchases required. All features are included in the free plugin.

= Can I use this without external services? =

Yes! The plugin works completely offline with local storage, or you can use your own FTP/SFTP servers.

= Is my data secure? =

Absolutely. All data stays on your servers or your chosen storage. No external services have access to your site data.
